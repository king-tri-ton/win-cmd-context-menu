# Windows CMD Context Menu

Простая, невесомая утилита, которая возвращает классическую «Командную строку» (cmd.exe) в контекстное меню Windows 10 / 11 по правому клику мыши.

## В чём фишка?
* **Ноль мегабайт в оперативной памяти** (работает через нативный реестр).
* Не требует зажатия клавиши `Shift`.
* Открывает консоль сразу в той папке, где вы вызвали меню.

## Установка

1. Перейдите во вкладку [Releases](../../releases) и скачайте архив `cmd-context-menu.zip`.
2. Распакуйте в любое место.
3. Нажмите правой кнопкой мыши на **`install.bat`** -> *«Запустить от имени Администратора»*.

## Удаление
Запустите `uninstall.bat` от имени Администратора.

## Поддержка Windows 11
В Windows 11 пункт появляется в разделе «Показать дополнительные параметры». Чтобы вернуть классическое полное меню...
